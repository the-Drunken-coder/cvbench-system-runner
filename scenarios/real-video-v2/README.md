# `real-video-v2`

Three public MEVA-derived, 30 FPS, full-frame multi-object tracking sequences. The checked-in scenario manifests and ground truth are exact public mirrors used by the catalog; runtime JPEGs are recreated below ignored `data/real-video-v2` and verified against `expected-frame-sha256.txt`.

There are no target hints, ignore rows, or scoreable ROIs. See `docs/real-video-sources.md` for licensing, ontology, upstream hashes, physical-ID corrections, and complete preparation commands.
